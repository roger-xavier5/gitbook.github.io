[苹果MacOS电脑使用vpn客户端上国外网站的方法]

##vpn推荐

Qv2ray客户端的介绍

QV2ray它一款Linux / Windows / macOS 跨平台 V2Ray 客户端 | 支持 Trojan-Go / Trojan / Vless / NaiveProxy / V2ray / SSR / SS | 使用 C++ / Qt5 开发 | 可拓展插件式设计的V2ray客户端。

Q2V2ray项目github地址：https://github.com/Qv2ray/Qv2ray/releases

1、MacOS电脑下载Qv2ray客户端

点击上面的地址，点开Assets10，找到macos-x64.dmg文件，点击下载

2、下载Qv2ray插件

Qv2ray 插件是 Qv2ray 的扩展，能让你使用更多功能特性。当前，Qv2ray 项目组共发布以下 6 个项目大佬维护的插件：

  • 点击下载QvPlugin-SS – 让你能在 Qv2ray 中启用 Shadowsocks SIP003 完整支持
  • 点击下载QvPlugin-SSR– 让你能在 Qv2ray 中使用 ShadowsocksR
  • 点击下载QvPlugin-Trojan – 让你能在 Qv2ray 中使用 Trojan
  • 点击下载QvPlugin-NaiveProxy – 让你能在 Qv2ray 中使用 NaiveProxy
  • 点击下载QvPlugin-Command– 当特殊事件发生时，运行任意指定的命令
  • 点击下载QvPlugin-Go – 让你能在 Qv2ray 中使用 Trojan-Go

3、Qv2ray客户端 & v2ray核心:

Windows客户端 Qv2ray（64位）    点击下载

Windows v2ray 核心（64位)             点击下载

MacOS Qv2ray 客户端                        点击下载

MacOS Qv2ray 核心                            点击下载

我们上面下载了MacOS Qv2ray客户端，这里就不用重复下载了。这里只需要下载MacOS Qv2ray核心文件即可。

安装Qv2ray软件

1、为了方便快速演示，我这里只下载了MacOS Qv2ray客户端 、MacOS Qv2ray 核心、 QvPlugin-SSR 和QvPlugin-Trojan 插件

2、双击Qv2ray的安装文件，是以.dmg结尾的，弹出小窗口后，将Qv2ray图标拖到右边来。

3、点击启动台，就可以看到QV2ray软件已经安装好了

4、双击打开Qv2ray软件，如果出现“无法打开qv2ray，因为无法验证开发者”这样的提示的时候，不要关闭这个窗口，我们点击桌面——系统偏好设置

5、接上一步，点击——安全性与隐私，然后点击左下角的那个锁，输入电脑的登录密码，点击解锁

6、点击取消

7、重新打开Qv2ray软件，会出现下图介面，点击——仍要打开

8、点击——打开

9、然后就打开Qv2ray软件了

我们在打开Qv2ray软件的时候有可能会出现几次8-11的情况，如果出现打不开的情况，就找到-安全性与隐私，点击允许就可以了。

10、设置中文语言，点击Qv2ray左上角的——Preferences

11、找到Language，点击下拉三角选择zh_CN，点击OK，就变成中文界面了。

安装Qv2ray插件

1、找到刚刚下载的SSR和Trojan插件，一起选中鼠标右键——拷贝

2、打开Qv2ray软件，点击左上角的——插件，点击——打开本地插件目录，在这个里面粘贴刚刚复制的两个插件

安装V2ray内核文件

1、双击v2ray内核文件压缩包，将它解压出来

2、打开Qv2ray软件，点击——首选项——内核设置，复制核心可执行文件路径

3、找到桌面的前往——前往文件夹

4、删除前往文件夹里的东西，粘贴刚刚复制的，删除后面的两项（vcore/v2ray）结果如下图：

5、在这个里面新键vcore文件夹

6、打开v2ray-macos内核文件夹

7、全选复制文件夹里所有内容

8、打开第5步刚刚建好的vcore文件夹，把所有复制内容粘贴进去

9、打开Qv2ray软件，点击——检查V2ray核心设置，这时候出现弹窗——无法打开“v2ray”,因为无法验证开发者 的提示，我们点击——取消

10、打开安全性与隐私，点击——仍然

11、点击——检查V2ray核心设置,会提示检查通过，点击OK OK  .如果检查不通过，那么这两个路径设置出了问题，检查并重新按教程设置这两个路径。

12、关闭软件，电脑右上角有一个图标，鼠标右键选择-Quit

13、在重新打开Qv2ray软件，会有如下图的提示，点击——取消。然后还是找到安全性与隐私，点击——仍然允许，就可以打开Qv2ray软件了。

14、点击插件，如果没显示安装的插件，关闭并退出Qv2ray软件，重新打开Qv2ray软件，点击——打开，打开，点击插件，就可以看到安装好的插件了。

这个软件在第一次安装的时候会老是弹出小窗口，我们就到安全性与隐私里点击允许即可。

使用机场订阅链接配置Qv2ray软件

1、我们这里以桔子云机场为例，打开桔子云机场后台，复制V2ray订阅链接或者SSR订阅链接，我们这里复制SSR订阅链接

2、打开Qv2ray软件，点击——订阅

3、分组名称随便填，点击——订阅设置，勾选此分组是一个订阅，订阅地址这里粘贴刚刚从机场复制的订阅链接，点击——Update subscription（更新订阅），点击OK

4、我们就能看到SSR这个里面有21个节点了，双击可以展开

5、选中其中的一个节点，鼠标右键选择——连接到此服务器

6、我们上油管youtube看一下4K视频
